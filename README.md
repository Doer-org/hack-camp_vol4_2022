## Getting Started

```docker-compose.env```をトップディレクトリに用意する

例
```
POSTGRES_USER=<USERNAME>
POSTGRES_PASSWORD=<PASSWORD>
PGPASSWORD=<PASSWORD>
POSTGRES_DB=<DBNAME>
TZ=<TIMEZONE>
```

<br>

```
docker-compose up -d
```
